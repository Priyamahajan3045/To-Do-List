# 📝 To-Do List App

A feature-rich Todo application built with React.js.

## 🚀 Features
- ✅ Add, Edit, Delete tasks
- 🔍 Search/Filter tasks
- 🌙 Dark Mode
- ☑️ Mark tasks as complete
- 📊 Task counter (Total, Done, Pending)

## 🛠️ Tech Stack
- React.js
- useState Hook
- CSS3

## 🏃 How to Run
1. Clone the repo
2. Run npm install
3. Run npm start
